Jak nastavit Supabase
1
Jdi na supabase.com, vytvoř projekt, zkopíruj URL a anon key z Settings → API
2
V Supabase otevři SQL Editor a spusť obsah souboru supabase-setup.sql
3
Vlož credentials výše a klikni Připojit
